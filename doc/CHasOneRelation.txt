
*CHasOneRelation*   >> |CActiveRelation| >> |CBaseActiveRelation| >> |CComponent|
=================

 CHasOneRelation represents the parameters specifying a HAS_ONE relation.


PROPERTY DETAILS
------------------------------------------------------------------------------

<through>				*CHasOneRelation::through*
 (string) the name of the relation that should be used as the bridge to this
 relation. Defaults to null, meaning don't use any bridge.


*CMysqlTableSchema*   >> |CDbTableSchema| >> |CComponent|
===================

 CMysqlTableSchema represents the metadata for a MySQL table.


PROPERTY DETAILS
------------------------------------------------------------------------------

<schemaName>				*CMysqlTableSchema::schemaName*
 (string) name of the schema (database) that this table belongs to. Defaults
 to null, meaning no schema (or the current database).

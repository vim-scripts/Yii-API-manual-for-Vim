
*COciTableSchema*   >> |CDbTableSchema| >> |CComponent|
=================

 COciTableSchema represents the metadata for a Oracle table.


PROPERTY DETAILS
------------------------------------------------------------------------------

<schemaName>				*COciTableSchema::schemaName*
 (string) name of the schema (database) that this table belongs to. Defaults
 to null, meaning no schema (or the current database).
